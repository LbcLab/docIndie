\subsection{Detailed Description}
This file defines version macros for Lib\+V\+LC. Those macros are primilarly intended for conditional (pre)compilation. To get the run-\/time Lib\+V\+LC version, use \hyperlink{group__libvlc__core_ga950b9828d1391f06fe56ce5d03483b9f}{libvlc\+\_\+get\+\_\+version()} instead (the run-\/time version may be more recent than build-\/time one, thanks to backward binary compatibility).

\begin{DoxyVersion}{Version}
This header file is available in Lib\+V\+LC 1.\+1.\+4 and higher. 
\end{DoxyVersion}
